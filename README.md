# GCU - Git Change User

Change git user easily and elegantly.

![demo gif](demo.gif)

## Install

Install Globally

```bash
npm install --global @owfdr/gcu
```

## Usage

Run GCU in your git repository.

```bash
gcu
```

## Features

- Easily switch between git users
- Add new git users
- Support global user and local user
